### Comandos de diretórios

```
                            *Comandos de navegação*                    

pwd // mostra o caminho atual que se está navegando 

ls // lista arquivos que estão no diretorio atual

ls *<tipo de arquivo> // lista todos os arquivos de mesmo tipo

cd ou cd ~ // muda para o diretorio home

cd - // muda para o diretorio anterior

cd <diretorio escolhido> // faz mudança de diretorios 

_________________________________________________________________________________

                                 *Comando rm*

rm <arquivo> // Faz a remoção de um arquivo unico porem a algumas outras formas de usar este mesmo arquivo:

rm <arquivo1> <arquivo2> ... // Para fazer a remoção de mais de um arquivo 

rm *<tipo de arquivo> // Usado para excluir arquivos de uma mesmo tipo ou de mesmo termino

rm -r <nome do diretorio> // Faz com que remova o diretorio especificado e todos os arquivos que se encontram nele

rm -i <nome do diretorio> // Faz com que remova o arquivos especificado porem não antes de sua confirmação 

_________________________________________________________________________________

                                 *Comando mv*

mv <arquivo> <diretorio> // Faz com que o arquivo escolhido seja movido para o diretorio escolhido 

mv <arquivo> <arquivo de mesmo tipo> // Muda o nome do arquivo primario para o escolhido

mv *<tipo de arquivo> <diretorio> // Faz com que os arquivos escolhidos sejam movidos para o diretorio escolhido 


/Sufixos do comando 'mv'

-i // realiza os comandos de forma mais interativa 

-n // não sobre escreve nenhum arquivo ja existente

-v // mostra passo a passo o processo que esta sendo feito

-f // faz o precesso a força sem a espera de uma confirmação
